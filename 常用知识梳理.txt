=====HTML=====
1.如何理解HTML语义化
        增加代码可读性；更利于搜索引擎SEO
2.块状元素&内联元素：
        display:block/table: 有div、h1(h2,h3...)、table、ul、ol、p 等
        display:inline/inline-block: 有span、img、input、button等

=====CSS=====
1.盒模型的宽度计算
        offsetWidth = (内容宽度 + 内边距 + 边框)，无外边距
                    = (width + padding + border)
        box-sizing:border-box 目标宽度为offsetWidth的值
2.margin纵向重叠问题
        相邻的margin-top和margin-bottom会发生重叠
3.margin负值的问题
        margin-top和margin-left负值，元素向上，向左移动
        margin-right负值，右侧元素左移，自身不受影响
        margin-bottom，下方元素上移，自身不受影响
4.BFC理解和应用：Block format context块级格式化上下文
        一块独立渲染区域，内部元素的渲染不会影响边界以外的元素
        形成BFC的条件：1.float不是none
                     2.position是absolute或fixed
                     3.overflow不是visible
                     4.display是flex，inline-block等
        常见应用：清楚浮动
5.float布局
        圣杯布局和双飞翼布局：
                三栏布局，中间一栏最先加载和渲染(内容最重要)
                两侧内容固定，中间内容随着宽度自适应
                一般用于PC网页
            要点：1.使用float布局
                 2.两侧使用margin负值，以便和中间内容横向重叠
                 3.防止中间内容被两侧覆盖，一个用padding一个用margin
6.手写clear fix
        .clearfix:after {
            content: '';
            display: block;
            clear: both;
        }
        .clearfix {
            *zoom: 1; // 兼容IE低版本
        }
7.flex布局
        常用语法：1.flex-direction 主轴对齐方向
                2.justify-content 主轴对齐方式
                3.align-items 交叉轴对齐方式
                4.flex-wrap 换行
                5.align-self grid或flex行中的元素对齐方式，能覆盖align-items值
8.定位
        relative依据自身定位
        absolute依据最近一层的定位元素定位
                定位元素：absolute，relative，fixed
                        body
9.居中对齐方式
        水平居中：1.inline元素：text-align:center
                2.block元素：margin:auto
                3.absolute元素：left:50% + margin-left负值
        垂直居中：1.inline元素：line-height的值等于height值
                2.absolute元素：top:50% + margin-top负值
                               transform:translate(-50%, -50%)
                               top,left,bottom,right = 0 加上margin:auto
10.图文样式
        line-height如何继承：1.继承具体数值
                           2.继承比例
                           3.百分比，继承计算出来的值(重点)：计算值是给定的百分比值乘以元素计算出的字体大小
11.响应式
        rem：相对于根元素的长度单位  1rem = 1 * 根元素值
                    media-query根据不同屏幕宽度设置根元素font-size
            rem弊端："阶梯性"
        vw/vh: 网页视口宽度/高度的1/100，vmax/vmin取两者最大/最小值
